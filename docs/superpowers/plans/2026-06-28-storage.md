# Storage Feature Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a per-channel key-value storage system with JSONB values, 30MB limit, dashboard UI, and executron sandbox API.

**Architecture:** Key-value store in PostgreSQL (one row per key per channel), GraphQL API reusing Variables permissions, capability module in executron sandbox (like twir:fetch), file-browser style dashboard tab.

**Tech Stack:** Go, pgx, squirrel, gqlgen, fx, TypeScript, Bun, isolated-vm, Vue 3, urql, TanStack Table, vee-validate, zod, shadcn-vue, Monaco editor
